QR Web Services

Description: These services were created for the QR feasibility study

To use the web services try:

1. Send a get request to /rest/api/recent

The server will return a list of the recent post requests that were send to "/rest/api/client"

2. Send JSON in a post request to /rest/api/client

This JSON will be saved in the cloudant database and will also be put in the recent request list.
